# Local Runtime

Zolem has two local execution modes:

- **Local runtime mode** (admin server) — create and modify profiles and listeners at runtime without restarting Zolem. See [admin-mode.md](admin-mode.md).
- **Fixed-listener mode** — start a single listener pinned to one provider and profile. See [fixed-listener.md](fixed-listener.md).

For fixture authoring (the `fixture` backend), see [fixture-authoring.md](fixture-authoring.md).

For recipes (no-egress Bubblewrap smoke tests), see [recipes/bubblewrap.md](recipes/bubblewrap.md).
